UTS SIG berbasis Web

Apa itu Sistem Informasi Geografis Berbasis Web?

SIG merupakan aplikasi Sistem Informasi Geografis (SIG) yang dapat diakses secara online melalui internet atau web. Pada konfigurasi SIGWeb ada server yang berfungsi sebagai Map Server yang bertugas memproses permintaan peta dari client dan kemudian mengirimkannya kembali ke client. Dalam hal ini pengguna/client tidak perlu mempunyai software SIG, hanya menggunakan internet browser seperti internet explorer, mozilla firefox, atau google chrome untuk mengakses informasi SIG yang ada di server (Sunaryo, 2015).

Sistem Informasi Geografis Web menjadi salah satu mata kuliah yang disediakan oleh Departemen Teknik Geodesi UGM untuk mahasiswanya sebagai mata kuliah pilihan. Mata kuliah Praktikum SIG Berbasis Web membahas mengenai bagaimana data dan informasi geospasial ditampilkan melalui internet, baik pada platform berupa desktop, mobile maupun media lainnya. Mata kuliah ini dimulai dari pembahasan mengenai cara kerja internet dan halaman web, konsep dan cara kerja client dan server pada WebGIS, arsitektur WebGIS serta desain dan evaluasi WebGIS.

Bagaimana internet bekerja?

Internet menggunakan kabel fiber optic dimana konduktornya lebih baik dibandingkan udara.

Adapun komponen-komponen dalam internet adalah sebagai berikut:

Client, adalah perangkat yang melakukan akses terhadap webside (orang/manusia yang mengakses juga dapat disebut sebagai client).

Server, adalah perangkat keras maupun lunak komputer yang berfungsi untuk menyimpan halaman web pada alamat tertentu.

Komunikasi antara client dan server dilakukan dengan bahasa dan protokol tertentu sehingga membentuk internet.
